**AndroidIDE Pro** is a cutting-edge Integrated Development Environment (IDE) designed specifically for Android devices. Built with **Kotlin** and **Jetpack Compose**, it brings a desktop-class coding experience to mobile, empowering developers to write, compile, and manage code on the go.

## üöÄ Features

-   **Advanced Code Editor**: 
    -   Syntax highlighting for multiple languages (Java, Python, Javascript, Kotlin).
    -   Line numbers, auto-indentation, and code completion.
-   **Integrated Terminal**: 
    -   Full shell access via internal terminal emulator.
    -   Execute commands and scripts directly.
-   **Robust File System**: 
    -   Seamless file management using Android's Storage Access Framework (SAF).
    -   Create, edit, and reorganize projects effortlessly.
-   **Git Integration**: 
    -   Clone repositories, commit changes, and push updates directly to GitHub.
-   **Compiler & Runtime**: 
    -   Local compilation support (where applicable) and script execution.

## üì¶ Installation

To build and run this project:

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/YOUR_USERNAME/AndroidIDE-Pro.git
    ```
2.  **Open in Android Studio**:
    -   Select "Open an existing Android Studio project" and navigate to the cloned directory.
3.  **Sync Gradle**:
    -   Allow Gradle to download dependencies.
4.  **Run**:
    -   Connect an Android device or start an emulator and click "Run".
